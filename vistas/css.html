Introduccion a CSS

CSS, siglas de Cascading Style Sheets (Hojas de estilo en cascada), es un lenguaje de diseno utilizado para describir la presentacion de un documento escrito en HTML. Con CSS, puedes controlar la apariencia de elementos HTML y darle estilo a tu pagina web.

Como se aplica CSS?

Hay tres formas principales de aplicar CSS a una pagina HTML:

  1. Estilos en lnea: puedes aplicar estilos directamente en la etiqueta HTML utilizando el atributo style. Por ejemplo: <p style="color: blue;">Este es un parrafo azul</p>.
  2. Estilos internos: puedes incluir los estilos CSS dentro de la seccion <style> en la cabecera del documento HTML, como se muestra en este ejemplo.
  3. Estilos externos: puedes crear un archivo separado con extension .css y enlazarlo con tu documento HTML utilizando la etiqueta <link>. Por ejemplo: <link rel="stylesheet" href="styles.css">.

Selector de elementos

Para aplicar estilos CSS a los elementos HTML, se utilizan selectores. Los selectores permiten identificar los elementos a los que deseas aplicar los estilos. Aqui hay algunos ejemplos de selectores comunes:

  • Selector de etiqueta: p selecciona todos los elementos <p> (parrafos).
  • Selector de clase: .highlight selecciona todos los elementos con la clase "highlight".
  • Selector de ID: #header selecciona el elemento con el ID "header".
